## **✅ Pasos básicos para realizar la homework**

Para iniciar, debes pararte sobre la carpeta **`01 - Excercises`**. Dentro de ella escribe los comandos:

```bash
npm install
npm test
```

Si deseas correr por test, puedes utilizar:

```bash
npm run test:01
```
